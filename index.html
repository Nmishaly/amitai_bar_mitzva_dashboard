📌 איך משיגים את הקוד הזה שוב?

  1. כנסו ל- Firebase Console והכנסו לפרויקט שלכם.
  2. לחצו על גלגל השיניים של ההגדרות (Settings) ליד Project Overview בצד שמאל למעלה -> Project settings.
  3. גללו לתחתית העמוד לקטגוריית Your apps.
  4. שם תראו את הריבוע שמכיל את קוד ה-SDK. העתיקו את כל הבלוק והדביקו כאן למעלה (המפענח החכם החדש יבין אותו בכל מקרה!).
