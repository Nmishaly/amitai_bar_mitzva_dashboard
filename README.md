# Amitai Bar Mitzvah Dashboard

A comprehensive dashboard application designed to manage and celebrate Amitai's Bar Mitzvah event.

## Overview

This project provides an interactive dashboard to organize, track, and display information related to Amitai's Bar Mitzvah celebration. It includes features for managing guest lists, event schedules, photos, and other important details for this milestone occasion.

## Features

- 📊 **Interactive Dashboard** - Centralized view of all Bar Mitzvah event information
- 👥 **Guest Management** - Track and manage guest lists and RSVPs
- 📅 **Event Scheduling** - Organize event timeline and activities
- 📸 **Photo Gallery** - Display and manage event photos
- 📱 **Responsive Design** - Works seamlessly across desktop and mobile devices
- 🎯 **Real-time Updates** - Live tracking of event details and confirmations

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager
- Git

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Nmishaly/amitai_bar_mitzva_dashboard.git
   cd amitai_bar_mitzva_dashboard
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server:**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Open your browser:**
   Navigate to `http://localhost:3000` to view the dashboard

## Usage

### Managing Guests

1. Navigate to the "Guests" section
2. Add new guests with their details
3. Track RSVP status
4. Export guest lists as needed

### Viewing Event Schedule

- Check the "Schedule" section for detailed event timeline
- Add or modify events and activities
- Set reminders for important milestones

### Photo Gallery

- Upload and organize event photos
- Create albums for different event moments
- Share gallery with family and friends
